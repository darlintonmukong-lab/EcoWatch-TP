# 🌍 EcoWatch : Plateforme de Collecte et d'Analyse Environnementale

## 📝 Présentation du Projet
EcoWatch est une application de **science citoyenne** développée pour le TP d'INF222. Elle permet aux utilisateurs de signaler en temps réel des incidents environnementaux (pollution plastique, décharges sauvages, etc.) afin de constituer une base de données exploitable pour l'analyse descriptive.

## 🚀 Fonctionnalités Clés
- **Collecte de données robuste :** Formulaire avec validation des types de pollution, niveaux de gravité et localisation.
- **Analyse Descriptive en temps réel :** Calcul automatique de la gravité moyenne et indicateurs d'état (Alerte/Vigilance/Stable).
- **Visualisation Dynamique :** Graphiques de répartition (Bar charts) et d'évolution temporelle (Line charts).
- **Fiabilité :** Système de persistance des données via stockage CSV et option d'exportation pour les analystes.

## 🛠️ Technologies Utilisées
- **Langage :** Python 3.x
- **Framework Web :** Streamlit (pour l'interface et le déploiement)
- **Analyse de données :** Pandas
- **Déploiement :** Streamlit Community Cloud

## 📊 Critères de Qualité Respectés
1. **Créativité et Imagination :** Concept original de monitoring environnemental participatif.
2. **Robustesse :** Gestion des erreurs de saisie et persistance des données.
3. **Efficacité :** Interface utilisateur fluide et calculs statistiques instantanés.
4. **Fiabilité :** Données sauvegardées et exportables en format standard (CSV).
